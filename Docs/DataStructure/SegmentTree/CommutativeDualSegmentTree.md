`using O = Monoid::Element`

`operation(u32 l, u32 r, const O& v)` 半開区間 $[l, r)$ に $v$ を合成(ACLでいう`composition`)
- $O(\log n)$

`operation(u32 i, const O& v)` 
- $O(\log n)$
- 未verify

`assign(u32 i, const O& v)` $i$ 番目の要素に $v$ を「代入」
- $O(\log n)$

`Operator operator[](u32 i)`
- $i$ 番目の要素を取得
- $O(\log n)$
